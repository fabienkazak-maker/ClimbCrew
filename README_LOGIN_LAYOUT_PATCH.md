# Patch ClimbCrew — amélioration mise en page login

## Modifications

- page de connexion recentrée et visuellement plus propre ;
- grand logo et titre mieux intégrés ;
- bloc "Compte par défaut" mis en valeur ;
- barre d’onglets plus lisible ;
- formulaires restructurés ;
- champs et boutons plus propres ;
- responsive amélioré sur mobile.

## Installation

Dézipper ce patch à la racine du projet local ClimbCrew, en remplaçant `App.jsx`.

Puis :

```bash
git add App.jsx README_LOGIN_LAYOUT_PATCH.md
git commit -m "Amelioration mise en page login"
git push
```

Ensuite sur Render frontend :

```text
Manual Deploy → Clear build cache & deploy
```
