# Correctif — page de login compacte

Ce patch rend la page de login :
- plus étroite ;
- centrée ;
- avec le logo et le texte centrés ;
- avec un logo réduit ;
- avec un bouton principal recentré.

## Installation

Dézipper à la racine du projet local ClimbCrew, en remplaçant `App.jsx`.

Puis :

```bash
git add App.jsx README_LOGIN_COMPACT.md
git commit -m "Page login compacte"
git push
```

Ensuite sur Render frontend :

```text
Manual Deploy → Clear build cache & deploy
```
