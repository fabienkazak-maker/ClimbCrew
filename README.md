# ClimbCrew — installation locale et serveur Linux

ClimbCrew est une application de gestion pour un club d'escalade : inscriptions aux séances, participants, voies, réalisations, progression et administration.

Le projet peut être lancé dans deux contextes :

- **en local Windows**, pour le développement et les tests ;
- **sur un serveur Linux**, derrière un reverse proxy HTTPS qui porte le certificat TLS et redir
